## 🗂️ **基本架構**

每筆記帳資料包含以下欄位：

| 欄位 | 說明 |
| --- | --- |
| 日期 | 消費發生日期，使用 YYYY-MM-DD |
| 品項 | 具體購買內容或服務名稱 |
| 金額（台幣） | 消費金額 |
| 原幣別／原幣金額／匯率 | 如有外幣消費則填入，否則預設為 TWD／1 |
| 付款方式 | 統一命名（見下） |
| 交易ID | `YYYYMMDD-HHMMSS`，時間戳記格式確保唯一性 |
| 明細說明 | 若需補充說明，可簡述情境或用途 |
| 分類 | 採用「橫向／縱向」一致邏輯（見下） |
| 專案 | 除固定項目外（見下），特定活動行程會另外專案紀錄，例如 `20250518-19 玉山南稜行` |
| 必要性 | 用於消費情緒與預算控管（見下） |
| 代墊狀態 | 標記是否為代墊或需支付他人款項 |
| 收款/支付對象 | 如代墊或收款需標明對象 |
| 附註 | 其他備註 |

## 🗂️ 各項目之說明彙整

| 分類名稱 | 說明 |
| --- | --- |
| 家庭 |  |

| `家庭／水果` | 單獨購買或預購的水果品項，如玉荷包、芒果、文旦等，通常具季節性或療癒性特質。 |
| `家庭／餐飲／＊＊餐` | 早餐、午餐、晚餐等；外出用餐或購買即食品項 |

| `家庭／飲品` | 液態型日常飲料，如現打果汁、冰沙、咖啡、茶飲等不屬於正餐的小飲品。 |
| `家庭／興趣與運動` | 運動用品、球類、樂器、拼圖、手作工具等具興趣性或休閒性質的家庭支出。適用於低頻但有方向性的生活品質提升項目。 |

| `家庭／服飾` | 家人共用、兒童衣物、家庭採購服飾等 |
| `家庭／通訊／網路費` | 家中寬頻、光纖、Wi-Fi 等固定上網費用 - 固定週期時屬年度方案 |
| `家庭／通訊／子女手機` | 子女使用之門號月租、上網吃到飽方案 - 固定週期時屬年度方案 |
| `家庭支出` | 玩具、紅包、雜項未分類類別 |
| 個人 |  |
| `個人／餐飲` | 公司聚餐、單人外食等 |
| `個人／服飾` | 自己的衣物、鞋子、飾品等個人用途 |
| `個人／通訊` | 個人手機月租、行動上網費 - 固定週期時屬年度方案 |
| 行程 |  |
| `行程／餐飲／＊＊餐` | 行程中的三餐花費 |
| `行程／點心` | 登山、戶外活動前後準備的點心補給 |
| `行程／門票` | 入園費、森林遊樂區等具「通行性質」的門票 |
| `行程／協作` | 登山協作人員的服務費（餐、帳篷、裝備搬運等） |
| `行程／裝備／租借` | 睡袋、帳篷等行程租賃支出 |
| 健康 |  |
| `健康／醫療／本人` | 本人健康檢查、藥品、就醫等 |
| `健康／醫療／家庭成員` | 先生、小孩或其他家人健康相關支出 |
| `健康／保險`  | 健康相關保險，如壽險、醫療險、失能險、癌症險、長照險等 |
| 教育 |  |
| `教育／子女` | 子女的課後班、教材等 |
| `教育／進修` | 成人自主進修、講座、學習活動（如：山難講座、野外急救課程、講習證照等） |
| 禮物 |  |
| `禮物／家庭` | 給家人或家庭聚會使用的禮物 |
| `禮物／外部贈送` | 伴手禮、送朋友、同事禮物等 |
| 交通 |  |
| `交通／接駁` | 行程中的火車、高鐵、登山行程的接駁車、共乘計程車等 |
| `交通／加油` | 私人汽車加油費用 |
| `交通／保險`  | 車輛強制險、車體險、第三責任險等 |
| `收款／代墊回收` | 回收的代墊金額 |

### 📌 **專案**

| 選項 | 說明 |
| --- | --- |
| `日常` | 日常生活中常規支出，不屬於特定活動或年度性項目。 |
| `家庭年度支出` | 如報稅、保險等家庭整體的年度性固定或特殊支出。 |
| `子女年度支出` | 子女相關的年度費用，如補習、課程、健康費用等。 |
| `紀念日／送禮` | 特定節日、生日或送禮用途。 |
| `健康檢查` | 本人或家人健康檢查專案用途。 |
| `登山行程` | 非特定登山行程準備之物品、裝備或課程 |
| `折扣／優惠` | 為特定折扣或促銷優惠而額外進行的採購支出。 |

---

### 📌 **必要性**

| 選項 | 說明 |
| --- | --- |
| `必要日常支出` | 日常基本開銷，例如三餐、衛生用品、醫療等必要支出。 |
| `想吃想買但合理` | 非必需但仍屬合理，例如生日慶祝、合理點心、家人想吃等。 |
| `療癒性支出` | 為提升情緒、滿足自我療癒而購買，如甜點、點心、舒壓用品等。 |
| `衝動購物（提醒）` | 明顯非必要或不預期消費，用以提醒反思之用。 |

---

### 📌 **代墊狀態**

| 選項 | 說明 |
| --- | --- |
| `無` | 本人支出，無代墊或待收待付情況。 |
| `不索取` | 本人代墊他人費用，但不會索取回款（如對家人支援）。 |
| `代墊` | 本人代為支付，需日後由對方付款給我。 |
| `需支付` | 他人代墊，本人需支付給對方。 |
| `已支付` | 本人已完成付款予對方（處理完成的需支付項目）。 |
| `已收款` | 已經收到對方的還款（處理完成的代墊項目）。 |
