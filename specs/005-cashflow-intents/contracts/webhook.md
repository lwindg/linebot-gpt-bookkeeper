# Webhook Contract: Cashflow Intents

本文件定義現金流意圖的 webhook 輸出格式，延續既有 CREATE 操作。

## CREATE Payload

```json
{
  "operation": "CREATE",
  "日期": "2026-01-08",
  "品項": "合庫提款",
  "原幣別": "TWD",
  "原幣金額": 5000,
  "匯率": 1,
  "付款方式": "合庫轉帳",
  "交易ID": "20260108-123456",
  "明細說明": "",
  "分類": "提款",
  "專案": "日常",
  "必要性": "必要日常支出",
  "代墊狀態": "無",
  "收款支付對象": "",
  "附註": "",
  "交易類型": "提款"
}
```

## 規則

- 雙筆記錄需使用相同的交易批次 ID（既有批次規則）
- `交易類型` 必須是 `提款` / `轉帳` / `收入` / `支出` / `折扣`
- 未提供付款方式時使用 `NA`
